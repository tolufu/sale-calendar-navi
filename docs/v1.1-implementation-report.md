# v1.1 実装レポート

最終更新: 2026-05-29

## 実装内容

- `WishItem` をv1.1形式へ拡張し、価格候補、価格内訳、参考リンク、最終確認日を保存できるようにした。
- localStorageの旧v1データを読み込み時にv2形式へ遅延移行する `migrateWishItem` を追加した。
- 実質価格計算を `calculateEffectivePrice(input)` に追加し、既存の数値引数形式も後方互換で維持した。
- 楽天URLのみ `scid=af_link_dummy` を付与する `convertRakutenAffiliate` と、merchant設定駆動の `buildAffiliateUrl` を追加した。
- 欲しいもの登録フォームに詳細入力の折りたたみ、5項目の価格内訳、リアルタイム計算、参考リンク複数保存を追加した。
- 欲しいもの一覧に希望価格、実質価格メモ、計算済み実質価格、参考リンク有無、最終確認日、手入力値の差額目安表示を追加した。

## 禁止事項チェック

- 外部EC、価格.com、楽天API等から価格・在庫・画像・レビューは取得していない。
- 商品画像は引き続き `public/images/placeholders/` の自作プレースホルダーのみを使う。
- 「最安値」「最安保証」等の断定・保証表現は追加していない。
- 参考リンクはURL保存のみで、自動取得や転載は行わない。
- 楽天アフィリエイト変換はダミーパラメータで、秘密情報は保存していない。
