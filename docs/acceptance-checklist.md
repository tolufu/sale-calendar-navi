## 横断チェック（全フェーズ共通）

- [ ] スクレイピング・外部価格自動取得・外部商品画像取得を行っていない。
- [ ] 「最安値」「最安保証」等の断定/保証表現がコード・コピーに無い。
- [ ] 公式サービスと誤認させるロゴ・文言を使っていない。
- [ ] Firebaseキー等の秘密情報をコミットしていない。
- [ ] ECは enum固定ではなく merchantId / merchants マスタで管理している。
- [ ] `npm run lint` / `npm run test` / `npm run build` が通る。
