# Amazon商品連携 TODO

v2ではAmazon商品検索の本実装は行わない。

## 理由

- Amazonアソシエイト/PA-APIの利用条件、審査、表示要件の確認が必要。
- APIキーや署名情報をブラウザへ出さないサーバー側実装が必要。
- 商品画像、価格、在庫表示は規約に沿ったキャッシュ/表示制約の確認が必要。

## 実装時の方針

- `AmazonProductSearchProvider` を実装し、UIからは楽天と同じ候補型で扱う。
- API呼び出しは必ずサーバー側ルート経由にする。
- 価格・画像・レビューのスクレイピングは行わない。
- 表示文言は公式サービスと誤認されないようにする。
